# AmtAssist

AI-powered browser assistant for German government forms and bureaucracy.

## Overview

AmtAssist helps residents, expats, students, and job seekers understand and complete German forms faster.

### Features

- Smart form detection
- AI field explanations
- Bureaucratic term translation
- Autofill recurring information
- Multi-language assistance
- Privacy-first design

## Target Platforms

- Chrome Extension
- Edge Extension
- Firefox Extension (planned)

## Roadmap

### MVP
- Field detection
- AI tooltips
- User profile autofill

### Phase 2
- Form validation
- Translation engine
- Government portal support

### Phase 3
- AI assistant
- Document analysis
- Personalized guidance
